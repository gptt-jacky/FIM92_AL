# 刺針感測器通訊系統

最後更新日期：2026-05-06   [V1.1](#版本紀錄)      [檔案連結](https://drive.google.com/file/d/1BNC2OZ81dBVmscCLuFWhVZjpk7nA73k8/view?usp=sharing) 

## **系統架構**

本系統由以下硬體組成：

・Antilatency（AL）定位系統：提供空間定位基礎設施，使用地墊系統  
・ALT 追蹤裝置（Alt Tracker）：裝設於各訓練設備上，回傳即時位置與旋轉資料  
・AL IO 模組（HW Extension Module）：連接各裝置的實體訊號腳位，讀取開關狀態並提供輸出控制

追蹤主機（Windows 電腦）透過 USB 接收 ALT 裝置與 IO 模組的原始資料，計算出 6DOF 定位結果（3 軸位置 \+ 3 軸旋轉），連同 IO 狀態整合為 JSON 格式，透過 WebSocket 即時推送給外部應用程式。

## **通訊規格**

| 通訊協定： | WebSocket（ws://） |
| :---- | :---- |
| **預設 Port：** | **8765（可在啟動時修改）** |
| **Frame 類型：** | **Text frame（UTF-8）** |
| **資料格式：** | **JSON** |

## **資料格式**

每筆 WebSocket message 為一行 JSON 字串（無換行）。

實際範例（6 台全連線）：

```json
{"trackers":[{"tag":"A","px":1.256012,"py":1.680032,"pz":0.000145,"rx":-0.710743,"ry":0.008621,"rz":-0.014023,"rw":0.703312,"io":"00031010"},{"tag":"B","px":0.000000,"py":0.000000,"pz":0.000000,"rx":0.000000,"ry":0.000000,"rz":0.000000,"rw":1.000000,"io":"00000000"},{"tag":"C","px":0.534210,"py":1.120045,"pz":-0.210034,"rx":-0.650123,"ry":0.012345,"rz":-0.008901,"rw":0.759012,"io":"10000000"},{"tag":"D","px":0.000000,"py":0.000000,"pz":0.000000,"rx":0.000000,"ry":0.000000,"rz":0.000000,"rw":1.000000,"io":"10000000"},{"tag":"E","px":0.912345,"py":1.750012,"pz":0.050678,"rx":-0.697612,"ry":0.008834,"rz":-0.010012,"rw":0.716023,"io":"00000000"},{"tag":"F","px":0.812000,"py":1.700056,"pz":0.060123,"rx":-0.701234,"ry":0.009012,"rz":-0.011234,"rw":0.712890,"io":"00000000"}]}
```

### **欄位定義：**

| tag | string | 裝置代號（"A" "B" "C" "D" "E" "F"） |
| :---- | :---- | :---- |
| **px / py / pz** | **float** | **位置（公尺）~~Tag B~~、Tag D 固定為 0** |
| **rx/ry/rz/rw** | **float** | **旋轉四元數 (x, y, z, w)。~~Tag B~~、Tag D 固定為 (0, 0, 0, 1\)** |
| **io** | **string** | **[8 字元 IO 狀態字串](#io-字串定義)**  |

## **IO 字串定義** {#io-字串定義}

io 欄位為固定 8 字元字串。Tag A 的 IOA3（位置 2）值為 0/1/2，IOA4（位置 3）值為 0–7；其餘裝置每個字元為 "0" 或 "1"。

**Tag A  刺針本體**

| 腳位 | 方向 | 功能 |
| :---- | :---- | :---- |
| IO1 | 輸入 | IFF 開關 |
| IO2 | 輸入 | 鎖定鍵 |
| **IOA3** | **輸出** | **震動器（0=OFF, 1=中震動, 2=強震動）** |
| **IOA4** | **類比輸入** | **天線 / BCU / 保險 decode（0–7，見下表）** |
| IO5 | 輸入 | 瞄準模組 |
| IO6 | 輸入 | 板機 |
| IO7 | 輸出 | 軟體允許後座力訊號 |
| IO8 | 輸入 | 後座力準備就緒 |

**IOA4 decode 對照：**

| 值 | 狀態 |
| :---- | :---- |
| 0 | none |
| 1 | BCU |
| 2 | 保險 |
| 3 | 天線 |
| 4 | BCU \+ 保險 |
| 5 | BCU \+ 天線 |
| 6 | 保險 \+ 天線 |
| 7 | BCU \+ 保險 \+ 天線 |

解讀範例：  
`"00000000"` → 待機（所有訊號 OFF）  
`"00200000"` → IOA3=強震動  
`"00031010"` → IOA4=天線, IO5=瞄準=ON, IO7=允許後座力=ON  
`"11131110"` → IFF=ON, 鎖定=ON, IOA3=中震動, IOA4=天線, IO5=ON, IO6=ON, IO7=ON

## **控制指令（Client → Server）**

### IO7 控制（Tag A 允許後座力）

```json
{"io7":"A1"}  →  Tag A IO7 設為 ON（允許硬體端觸發後座力）
{"io7":"A0"}  →  Tag A IO7 設為 OFF（禁止硬體端觸發後座力）
```

### IOA3 控制（Tag A 震動器）

```json
{"ioa3":"A0"}  →  Tag A IOA3 設為 OFF（停止震動）
{"ioa3":"A1"}  →  Tag A IOA3 設為中震動
{"ioa3":"A2"}  →  Tag A IOA3 設為強震動
```

### 操作範例

場景一：停止 → 中震動

```json
發送：{"ioa3":"A1"}
結果：IOA3=中震動
```

場景二：停止 → 強震動

```json
發送：{"ioa3":"A2"}
結果：IOA3=強震動
```

場景三：震動中 → 停止

```json
發送：{"ioa3":"A0"}
結果：IOA3=OFF
```

場景四：中震動 → 強震動

```json
發送：{"ioa3":"A2"}
結果：IOA3=強震動
```

## **WebSocket Server 設定**

Server IP：執行 RunDataServer.bat 後畫面會自動列出本機 IP  
Port：8765（預設，啟動時可修改）  
完整連線位址：ws://\<Server IP\>:8765

Port 修改方式：執行 RunDataServer.bat 後，出現提示時輸入自訂 Port 號，直接按 Enter 則使用預設值 8765。

## **IO 訊號監控工具（RunMonitor.bat）**

提供 Console 即時 IO 訊號監控，測試訊號用。直接在追蹤主機上執行即可。  
啟動方式：雙擊 RunMonitor.bat，Console 視窗會自動開始顯示。  
用途：  
・訊號測試：按下實體按鍵（板機、IFF 等），觀察對應 IO 是否即時變化  
・IO 輸出測試：使用鍵盤快捷鍵（A/O）控制 IO7 輸出，確認硬體反應  
・追蹤狀態確認：觀察 Stability 是否達到 S:2（6DOF 正常追蹤）
